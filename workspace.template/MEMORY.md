# Memory

Durable facts and preferences. Your assistant reads this at the start of every
new conversation, and appends to it when you ask it to remember something.

This file is yours. Edit it freely — it is plain markdown, and it is the single
source of truth for what your assistant knows about you. Nothing is hidden
anywhere else.

Keep it modest. Everything here is loaded on every new thread, so it costs
tokens and attention on each conversation. Facts that matter for a week belong
in `notes/` instead.
